# Customer Onboarding & Marketing Engagement Analysis

## Aim
The aim of this project is to analyze customer signup data to understand onboarding trends, customer demographics, and marketing engagement.

## Dataset
- customers.csv
- Contains customer information such as age, country, signup date, and marketing opt-in status.

## Analysis Performed
- Customer signup trends (daily & monthly)
- Age group and country distribution
- Marketing opt-in analysis
- Business insights from data

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Customer signups show clear growth trends over time.
- Certain age groups are more responsive to marketing.
- A few countries contribute most customers.

## Conclusion
This analysis helps businesses improve customer onboarding and targeted marketing strategies.
